\begin{frame}{Corruption model for graph simulations}
\begin{enumerate}
    \item Sample an uncorrupted graph from an SBM and \(m = \lfloor \gamma n \rfloor\) nodes to be the outliers.
    \item For each outlier \(i = 1, \dotsc, m\) and each \(k = 1, \dotsc, K\), draw a new connection probability between that node and nodes in community \(k\) using a Beta distribution, \textit{i.e.}, \(\tilde{\Gamma}_{ik} \sim \mathcal{B}(\alpha, \beta)\).
\end{enumerate}

Here, \(\alpha\) and \(\beta\) are chosen so that \(\mathbb{E}[\tilde{\Gamma}_{ik}] = \Gamma_{z(i) k}\) and that the variance is the greatest possible (with the constraint that \(\tilde{\Gamma}_{ik} \in [0, 1]\)).
\end{frame}
